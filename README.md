# Descrição:
O presente projeto tem como objetivo propor uma solução tecnológica para suprir a carência de gestão de
vendas para a empresa de eventos Casa de Festas Ilha Mágica Ltda. A ausência de um controle
estruturado impacta diretamente a margem de lucro, ocasionando perdas de oportunidades, baixa
eficiência no acompanhamento de negociações e dificuldade na fidelização de clientes.
Diante desse cenário, a implementação de um sistema de gestão de vendas com ETL (Extract, Transform,
Load), associado a um Data warehouse, apresenta-se como alternativa estratégica, capaz de centralizar
e organizar as informações comerciais desde o primeiro contato com o cliente até o fechamento do contrato
e o pós-venda. Com isso, busca-se não apenas aumentar a produtividade e a eficiência dos processos
internos, mas também proporcionar maior qualidade no atendimento e previsibilidade para a gestão
organizacional.
